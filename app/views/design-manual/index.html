Introduction

The Calculate release dates service uses sentence and offence information from NOMIS to automatically and accurately calculate prisoner release dates.

This design manual gives an overview of the service, including how each page works, the patterns used and user research gathered.

Service overview

Service entry points

Users can access the Calculate release dates service from two entry points. The service can be used as a standalone site, by using the service URL. This takes them directly to the search function to look up a person, to then calculate their release dates.

Alternatively, the service can be accessed from Digital Prison Services, by searching for a person and selecting a call to action on the Prisoner Profile.
