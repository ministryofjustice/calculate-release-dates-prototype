Introduction

The Calculate release dates service uses sentence and offence information from NOMIS to automatically and accurately calculate prisoner release dates.

This design manual gives an overview of the service, including how each page works, the patterns used and user research gathered.

Service overview

Service entry points

Users can the Calculate release dates service from 2 entry points.

Users can use the service as a standalone site, by using the service URL. This takes them directly to search function to lookup the person they want to calculate release dates for.

Users can also access the service from Digital Prison Services by searching for a person and selecting a call to action on the Prisoner Profile.

Users must enter sentence and offence information for the person they want to calculate release dates for into NOMIS before using the service.
