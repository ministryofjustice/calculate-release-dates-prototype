What this page does

This page helps users understading what the service does, and what they must do before using it.

In addition to this, this page also informs users of the calculation lookup service where they can view previous calculations completed by Calculate release dates.

Users can also check which sentence types are supported by the service from here.

User research

Page analytics show that a low proportion of users are using the calculation lookup service, or viewing the supported sentence types from this page.

Future considerations

Further research may need to be carried out to understand how often users need to use the calculation lookup service or check supported sentences.

In future, we might want to give the calculation lookup service more prominance as we migrate away from NOMIS. We might also need to consider replacing this page as Calculate release dates integrates with the Digital Warrant Folder service.
