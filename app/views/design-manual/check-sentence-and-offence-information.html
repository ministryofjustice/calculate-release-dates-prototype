Check sentence and offence information

Stable

Help users make sure the information used in the calculation from NOMIS is correct.

What this page does

On this page, users review the court case, sentence and offence information that has been entered into NOMIS and used in the calculation.

Before using the Calculate release dates service, users must enter sentence and offence information into NOMIS. Because of usability problems with NOMIS, this information can be difficult to enter and view.

By laying out information clearly, the Calculate release dates service allows users to ensure that all information used in a calculation is present and correct.

If any information is incorrect, users must correct it in NOMIS and reload this page to review changes.

Adjustments

Adjustments that are to be applied to a calculation are also displayed on this page. It shows a summary of all adjustments, as well as more detailed information about some adjustments such as remand, which may affect the outcome of a calculation.

Include an ERSED date

Currently there is no functionality in other services to identify a person who requires an ERSED (Early release scheme eligibility date) as part of their calculation. Until functionality in other services allows this, users must select to include an ERSED date in a calculation on this page using a checkbox.

User research

Multiple versions of this page have been tested with users in user research and during the service's private beta phase.

User research has shown us that users understand the page and are able to use it.

We have heard from users that the layout of information makes it easy to read and understand, giving them confidence that the correct information is being used in the calculation.

Usability testing found that it is important to display a “0” when there are no adjustments of a particular type. This helps reassure users that information has not been missed and has been entered correctly.

In usability testing of previous iterations of the adjustment section, users wanted to see more detailed information about remand dates. This information is used by users to understand which sentence remand is applied to.

Future considerations

As the Digital warrant folder service replaces NOMIS for the input of court cases, sentence and offence information, the purpose of this page will change. We believe there will still be a user need to review the information used in a calculation, but will need to consider how this page integrates with that service.

Although initial user research suggests users are able to include an ERSED in the calculation, and understand the meaning of this, we must monitor this component over time. As ERSEDs are only applicable in a minority of calculations, users might develop a behaviour where they skip this, as it's not relevant in most cases.
