Start page

Stable

Help users start using the service and understand what it does.

What this page does

This page helps users understand what the service does, and what they must do before using it.

In addition to this, this page also informs users of the calculation lookup service where they can view previous calculations completed by Calculate release dates. Users can also check which sentence types are supported by the service from here.

User research

Page analytics show that a low proportion of users are using the calculation lookup service, or viewing the supported sentence types from this page.

User research shows that users do not notice the Calculation lookup service link.

Accessibility

Findings of an accessibility audit showed that the colour contrast of the Start button used on this page doesn’t meet the criteria for WCAG 2.2. It was also noted that it might not be clear to some visually impaired users what the purpose of the “Start now” label is.

This page uses the Start using a service design pattern from the GOV.UK design system. We have reported findings about the colour contrast to the Design system team and are exploring alternative designs that will be more accessible.

Future considerations

Further research may need to be carried out to understand how often users need to use the calculation lookup service or check supported sentences.

In future, we might want to give the calculation lookup service more prominence as we migrate away from NOMIS. We might also need to consider replacing this page as Calculate release dates integrates with the Digital Warrant Folder service.

We have designed an initial version of a landing page that could replace the start page in future versions of the service. This is currently awaiting usability testing.
