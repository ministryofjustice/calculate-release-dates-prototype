Search for a person

Stable

Lookup a person to calculate release dates for.

What this page does

When a user is using the standalone version of Calculate release dates (not through Digital Prison Services), they can use search to find a person to calculate release dates for.

Users can search for a person by using their prison number, first name, last name or any combination of these.

The search results show a list of matching people from NOMIS including their prison number and date of birth. This information helps users identify the correct person in situations where there may be multiple people with the same name.

From here users can select a person to calculate their release dates.

User research

This page has tested well with users in user research.

Future considerations

In future we might consider a single search field that allows users to enter multiple search terms. This will be more aligned to other services across GDS.
