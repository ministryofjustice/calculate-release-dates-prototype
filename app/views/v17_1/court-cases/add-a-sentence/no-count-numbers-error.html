Add offences Enter count numbers for multiple consecutive sentences

You must enter count numbers for offences when there is more than one consecutive sentence listed on the same warrant.

Why count numbers are important
Count numbers are given to each offence on a court case. Sometimes these are also called Offence numbers.

Some warrants don't show count numbers, but they are needed to show the correct order of consecutive sentences. Entering count numbers incorrectly could lead to inaccurate sentence calculations.

What you need to do

  • Review offences added for this appearance
  • Edit each sentence to add the count number

If there are no count numbers on the warrant

If the warrant doesn't have count numbers, you can either:

  • Check other documents such as the indictment or prison court register relevent count number for each offence
  • If you have access, check Common Platform
  • Contact {{ data.appearance['court-name'] }}
